## Setting up

To run Lavita AI Jobs, you need to first install the [Theta Edge Node](https://docs.thetatoken.org/docs/setup-theta-edge-node). After installing the Edge Node, go to **Settings**, enable **Lavita (AI/ML) Jobs**, and save your settings. Once saved, your device will start receiving AI jobs. See the picture below.
